# EVEEX-FPGA
Hardware implementing of the EVEEX project.
The goal here is to implement EVEEX algorithm onto an fpga to process image coming from OV7670 CMOS camera.

## material 
### Software used
Vivado Hlx 2020.1  
Nexys4-master xdc file  
digilent board git files  

### hardware used 

Nexys 4 board
OV7670 cameras
